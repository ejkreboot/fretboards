Color Circles With Names

This is the a-minor hexatonic (blues) scale with root notes colored red and the "blues notes" colored blue.
Note that when creating a fretboard diagram, you can turn on note names and specify both the circle fill colors and circle radius. If you provide less than seven colors for the fill and note names, your colors will be recycled. Here is an example 
      var fb = fretboard.Fretboard(
        {
            radius: 8,
            showNames: true,
            fillColors: ["red", "white", "white", "blue", "white", "white", "white"],
            nameColors: ["white", "black", "black", "white", "black", "black"],
        })
        fb.add("a minor-blues").paint();
